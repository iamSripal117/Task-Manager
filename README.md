A simple and secure Task Management Web Application built using ASP.NET Core MVC, Entity Framework Core, and Identity Authentication.
This project allows users to register, log in, and manage their personal tasks securely.

**Features**
*1. User Authentication*

User registration & login using ASP.NET Core Identity

Email confirmation enabled (RequireConfirmedAccount = true)

Secure password and session handling

*2. Task Management*

Add new tasks

View task list

Edit/update tasks

Delete tasks

*3. Database Integration*

Database built using Entity Framework Core

SQL Server LocalDB used through the connection string in appsettings.json

*4. MVC Architecture*

Clean separation using Models, Views, and Controllers

Razor pages enabled for Identity UI

**Technologies Used:**
*Technology	Purpose*
ASP.NET Core 8/7	Backend framework
Entity Framework Core	ORM & database operations
SQL Server LocalDB	Development database
ASP.NET Identity	Authentication & Authorization
C#	Primary programming language
Razor Pages	Identity UI rendering
